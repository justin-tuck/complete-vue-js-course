# Vue JS 

Course to help learn Vue JS

## Set up
[Vue JS](https://vuejs.org/)

Create a Vue App by following the steps with this command:
```sh
npm create vue@latest
```

Then install dependencies and run your app:
```sh
cd <your-project-name>
npm install
npm run dev
```
